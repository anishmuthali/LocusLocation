Introduction

"Magic" is a fully responsive admin web App built with Bootstrap & AngularJS. It uses Sass CSS which makes it easy to modify.

How to Run it?

You'll need to put the files on a server to run it.

So you can upload the "dist" folder to your server to run it. Or you can build a server locally.

Then how to build a server locally?

You can build a locally server using WAMP or MAMP (if you uses PHP). Also it's pretty easy to build one with Python or Ruby (if you prefer Python or Ruby). And here we'll build a local server with NodeJS

Preparation:

You'll need to install Node.js (NPM comes along with it) and node-static (an NPM package)

Steps:

It's easy if you are familiar with Node. Download Node.js here then install it, and install node-static with one command line

$ npm install -g node-static

The usage of node-static is simple. Just enter the "dist" directory with command line

$ cd dist/

and type:

$ static

It'll return something like this: 

serving "." at http://127.0.0.1:8080

Then you can view the template on http://127.0.0.1:8080 with your browser

How to Use It? (Sass Version, The Default)

In order to do development with it, first you'll need have the following tools installed:

  • Node.js (NPM comes along with it), just download and install it, that's it, don't be afraid.
  • Yeoman , Bower, Grunt-cli for modern workflow.
  • Ruby (if you are using a Mac, then you already have it) , Sass , Compass for Sass CSS.

    • And you'll need to know how some basis about them, click the links to learn more if you not familiar with these yet. If it's your first time using these tools, it's probably too complex or even overwhelming to you. You'd better find someone to help you.

    If you already know how to use them, then just do a few things to start developing:

    1. Enter "Magic" folder with your command line tool.

    2. Install all the NPM packages (for development) with NPM: 

    $ npm install

    3. Install all the front end packages with Twitter Bower:

    $ bower install

    4. Then type grunt serve and you are ready to go.

    $ grunt serve

    By the way, if you want to build the distribution (production) version, simply run

    $ grunt serve:dist

    All the files for development are under ./Magic/client/ folder.

    Layouts

    There's two ways to change the default layout. (1) Change "index.html" file. (2) Change "app.controller.js" file inside "./client/scripts/core" folder.

    Both ways are easy. Here we'll talk about the second way since it's easier. Open "app.controller.js" file, you'll find following code. You can change the value to update default layout.

    $scope.admin = {
    layout: 'wide',                                 // 'boxed', 'wide'
    menu: 'vertical',                               // 'horizontal', 'vertical', 'collapsed'
    fixedHeader: true,                              // true, false
    fixedSidebar: true,                             // true, false
    pageTransition: $scope.pageTransitionOpts[0],   // unlimited
    skin: '12'                                      // 11,12,13,14,15,16; 21,22,23...
};

    For example you want set "boxed layout" as default, then you can update code like so:

    $scope.admin = {
    layout: 'boxed',                                // 'boxed', 'wide'
    menu: 'vertical',                               // 'horizontal', 'vertical', 'collapsed'
    fixedHeader: true,                              // true, false
    fixedSidebar: true,                             // true, false
    pageTransition: $scope.pageTransitionOpts[0],   // unlimited
    skin: '12'                                      // 11,12,13,14,15,16; 21,22,23...
};

    Done :)
